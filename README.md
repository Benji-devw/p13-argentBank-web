# ArgentBank

## Project Description
ArgentBank is a web application for managing personal bank accounts. Users can view their account balances, transaction history, and perform various banking operations.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/p13-argentBank-web.git
    ```

2. Navigate to the project directory:
    ```bash
    cd p13-argentBank-web
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the development server:
    ```bash
    npm start
    ```

2. Open your browser and navigate to `http://localhost:3000`.

3. Login

    ``` bash
    email: 'tony@stark.com',
    password: '123',
    ```
